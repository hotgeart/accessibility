## Accessible menu

https://github.com/hotgeart/accessible-menu

Don't forget to add aria-current (for Drupal) :

```js
// add aria-current
if($('#menu-main .menu .is-active').length){
  $('#menu-main .menu .is-active').attr('aria-current', 'page');
}
```
